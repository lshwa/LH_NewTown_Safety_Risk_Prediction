# LH_NewTown_Safety_Risk_Prediction
(LH) 신도시 교통 인프라 데이터 분석을 통한 3기 신도시 어린이 등 취약계층 교통안전 대안제시

## 🚦 신도시 보행사고 위험도 사전 예측 모델 (Preemptive Traffic Risk Modeling) 🚦

> 🏆 LH 스마트시티 공모전 제출 프로젝트

---

### 프로젝트 개요

기존 교통안전 분석은 **과거 사고 데이터**에 의존하지만,  
하남교산과 같은 **신도시(계획 도시)**는 사고 이력이 존재하지 않습니다.

본 프로젝트는 이러한 한계를 해결하기 위해,  
**도로 구조와 보행·교통 특성을 기반으로 사고 위험을 사전에 예측하는 모델**을 구축하였습니다.

---

### 핵심 아이디어

- 사고 데이터가 없어도 적용 가능한 **구조 기반 위험도 모델링**
- **격자 기반 공간 분할 (100m / 200m)**을 통한 데이터 통합
- 도로·교통·보행·시설 데이터를 결합하여 변수 생성
- **Random Forest → 변수 탐색**
- **Negative Binomial 회귀 → 최종 모델링 (과산포 반영)**
- **Interaction 변수**를 통해 보행노출 × 도로환경 효과 분석
- **CMF(Crash Modification Factor)**를 활용한 정책 효과 정량화

---

### 데이터 구성

- OpenStreetMap 기반 도로 네트워크
- 교통 데이터 (교통량, 평균속도, 혼잡도)
- 보행 노출 데이터 (유동인구, 어린이 활동)
- 교통 안전 시설
  - 횡단보도
  - 버스정류장
  - CCTV
  - 과속방지턱
  - 어린이보호구역
- 기존 도시 사고 데이터 (모델 학습용)

---

### 분석 방법론

1. 공간 격자 생성 (100m → 200m 집계)
2. 변수 생성
   - density 변수 (count / road_length)
   - 보행 노출 proxy
   - interaction 변수
3. 구조적 zero 제거
4. 모델 비교
   - Random Forest (변수 중요도 탐색)
   - Poisson 회귀
   - Negative Binomial 회귀
5. 모델 선택 기준
   - 과산포 (dispersion)
   - AIC 비교
6. 정책 효과 분석
   - CMF 기반 사고 감소 효과 추정

---

### 주요 분석 결과

- 사고 위험은 단순 교통량이 아니라  
  **보행 노출 × 도로 구조의 상호작용**에서 결정됨
- 고위험 지역은 단순 시설 부족이 아니라  
  **교통 상호작용 밀도가 높은 구간**
- 버스정류장 밀도, 차로 구조, 혼잡도가 주요 위험 요인
- 안전시설은 단순 효과보다  
  **고위험 구간에 집중 배치된 대응 효과**가 나타남
- 과속방지턱은 어린이 노출이 높은 지역에서  
  **위험 완화 효과 (interaction)** 확인

---

### 하남교산 적용

- 사고 데이터가 없는 신도시에 모델 적용
- 생성 결과:
  - 200m 격자 사고 위험도 지도
  - 100m 격자 취약도 지도
- 이를 통해 **사전 위험 구간 식별 및 대응 가능**

---

### 정책적 활용

- 위험도 기반 교통안전 시설 우선 설치
- 어린이 보행 취약 구간 식별
- CMF 기반 정책 효과 정량 평가
- **데이터 기반 신도시 안전 설계 지원**

---

### 결론

본 연구는 단순 사고 분석을 넘어,

> **사고 발생 이후 대응이 아닌,  
도시 설계 단계에서 위험을 사전에 탐지하고  
정책 효과까지 연결하는 데이터 기반 안전 설계 프레임워크를 제시한다.**
